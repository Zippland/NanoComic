# NanoComic 漫画分镜生成器

一个面向漫画创作的分镜生成项目：根据用户输入自动调研涉及的人物、事物与场景，并最终输出按分镜逐格的详细漫画脚本（包含镜头/构图、角色动作与表情、穿着、环境与道具、对话及语气等），不输出调研报告。

## 使用提示（简）

- 后端需配置环境变量 `GEMINI_API_KEY`
- 开发启动：`make dev`

运行后，输入你的创作主题或需求，系统将完成必要调研并输出细化到每一格的漫画分镜脚本。
